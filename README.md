# gentle-ai-community-landing

Community landing repo for **GENTLE-AI**.

## Goal

Build one shared landing page by combining the best ideas from the available references:

- `repo-yo/`
- `repo-fabri/`
- `repo-gerardo/`
- Becker public landing: https://gentleai.vercel.app/

## Current status

- `landing-context.md` contains the synthesis of the imported references.
- The cloned repos are kept here only as context snapshots.
- Community governance files are being added now.

## Community rules

- `main` stays protected.
- All changes go through PRs.
- PRs must link an approved issue.
- PRs must use exactly one `type:*` label.
- Copy should stay factual and consistent with the product.

## Governance files

- `CONTRIBUTING.md` — contributor workflow
- `GOVERNANCE.md` — branch protection and PR policy
- `.github/PULL_REQUEST_TEMPLATE.md` — PR structure
- `.github/ISSUE_TEMPLATE/` — issue templates

## Start here

1. Read `landing-context.md`
2. Read `CONTRIBUTING.md`
3. Open an issue for the change
4. Work in a branch named `type/description`
5. Open a PR using the template

## Notes

This repo is intentionally community-first: simple rules, clear review flow, and no direct pushes to `main`.
